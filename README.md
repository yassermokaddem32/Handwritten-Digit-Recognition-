# 🔢 Handwritten Digit Recognition

A Python application that recognizes handwritten digits using a trained deep learning model (e.g., MNIST dataset).

## 🚀 Features

* Recognizes digits (0–9) from images
* Uses trained neural network model
* Fast and accurate predictions
* Simple and easy to use

## 🛠️ Technologies

* Python
* TensorFlow / Keras
* NumPy
* OpenCV (optional)

## ▶️ How to Run

```bash id="d83kls"
pip install tensorflow numpy opencv-python
python project5_digit_recognition.py
```

## 📌 Description

This project uses a trained model to classify handwritten digits. It demonstrates image preprocessing, model inference, and deep learning fundamentals.
